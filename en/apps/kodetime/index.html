Kodetime

Your AI estimation companion

macOS 14+
Coming Soon

Kodetime is a native macOS application that uses Claude AI to analyze software projects and provide accurate time estimates for development tasks. Stop guessing, start estimating.

Features

  • AI-Powered Estimates - Claude AI analyzes tasks based on your project context
  • Project Management - Manage multiple software projects with auto-detected tech stack
  • GitHub/GitLab Integration - Import issues directly via OAuth
  • Detailed Breakdown - Subtasks, complexity factors, and confidence levels
  • Ticket Management - Create, organize, and track tickets with 6 types and statuses
  • Estimation History - Track accuracy with estimated vs actual time
  • Native macOS - Built with SwiftUI for a seamless Mac experience

Currently in development for macOS. Stay tuned for updates.
