Accessibility

ClipCard Pro

Our Commitment

At ClipCard Pro, we believe everyone deserves secure, easy access to their sensitive data. We are committed to ensuring that our app is usable by as many people as possible, including individuals with visual, motor, auditory, and cognitive disabilities. Accessibility is not an afterthought — it is built into every feature from the start.

VoiceOver — Full Support

ClipCard Pro offers comprehensive VoiceOver support throughout the entire app. Every interactive element includes descriptive labels, hints, and values so screen reader users can navigate with confidence.

  • Semantic traits (headers, buttons, links, selected states) for intuitive navigation
  • Combined accessibility containers for efficient swipe-through navigation
  • Decorative elements are hidden from VoiceOver to reduce clutter
  • Privacy-aware labels: sensitive data like card numbers are read as "ending in 1234" instead of the full number
  • Automatic secure information mode activates when VoiceOver is detected
  • Screen change announcements keep users oriented during navigation

Dynamic Type — Full Support

The app fully respects your preferred text size settings. Over 30 size metrics across the interface scale with Dynamic Type, and custom fonts are configured to scale properly via Font.scaledCustom(). Layouts automatically adapt to accommodate larger text sizes without breaking.

Reduce Motion

When the Reduce Motion setting is enabled on your device, all animations are removed. Page transitions, card expansions, theme selection effects, and other motion across 6+ major views are replaced with instant state changes for a comfortable experience.

Reduce Transparency

When Reduce Transparency is active, gradient backgrounds are automatically replaced with solid colors. This improves readability and reduces visual complexity for users who prefer higher contrast interfaces.

Dark Mode

ClipCard Pro includes a complete theme system with light and dark mode variants. Multiple themes are available, and every theme provides full dark mode support. The app follows your system appearance preference automatically.

Haptic Feedback

A complete haptic feedback system provides tactile confirmation for key actions. Success, error, warning, and selection haptics help users understand the result of their interactions — saving a clip, copying data, or deleting an entry — without relying solely on visual cues.

Privacy-Aware Accessibility

Security and accessibility work together in ClipCard Pro. The app uses intelligent masking by data type — credit cards, phone numbers, crypto wallets, and URLs — so that VoiceOver never reads complete sensitive information aloud.

  • Granular masking control per clip type available in Settings
  • Automatic secure mode when assistive technologies are active
  • Sensitive data is protected without sacrificing usability

Biometric Authentication

Face ID and Touch ID are fully integrated with accessible prompts. The authentication flow works seamlessly with assistive technologies and provides clear, localized prompts so every user can unlock their vault securely.

Localization

The interface is fully localized in Spanish, including all accessibility labels. VoiceOver users can navigate the app entirely in their preferred language with accurately translated descriptions and hints.

Feedback

We are constantly working to improve the accessibility of ClipCard Pro. If you experience any difficulty using the app with assistive technologies, or if you have suggestions on how we can improve, please contact us at: support@wilderlopezelias.com
