HappySells

Your currency exchange assistant, always visible

iOS 17+ iPadOS macOS watchOS Coming Soon
App Store — Coming Soon

HappySells converts any currency or cryptocurrency to your local currency in real time. It supports 240+ countries with automatic region detection and specialized data sources per market. Deeply integrated with the OS: interactive widgets, Dynamic Island, Apple Watch complications, and macOS MenuBar. Free with optional Pro subscription.

Features

  • Real-Time Conversion - Fiat (USD, EUR, CAD, MXN) and crypto (BTC, USDT, TRX) with live rates
  • Offline-First - Cached rates work without internet; freshness indicator (green/yellow/red)
  • Transaction History - Full history powered by SwiftData with search and filters
  • Share Sales - Share as text or visual card with watermark
  • Privacy Mode - Hide all numbers with a single tap
  • Auto Country Detection - Automatic region selector with 240+ country support
  • Configurable Base Currency - Set your preferred base currency
  • Premium Sound & Haptics - Immersive sound effects and haptic feedback
  • Smart Refresh Limits - Intelligent rate refresh to conserve bandwidth

Pro — $1.99/mo or $14.99/yr

  • Unlimited Widgets - Home Screen, Lock Screen, and StandBy widgets
  • Live Activities - Dynamic Island with real-time rate tracking
  • macOS MenuBar - Quick access via MenuBarExtra
  • Trend Charts - Rate history and trends powered by Swift Charts
  • Price Alerts - Get notified when rates hit your target
  • Unlimited History + CSV Export - Full transaction history with export
  • Earnings Dashboard - Profit tracking for traders
  • watchOS Complications - Multiple complications for Apple Watch

Data Sources

  • Cuba - elToque API (informal market rates)
  • Global Crypto - CoinGecko API
  • Global Fiat - ExchangeRate API

Built with 100% native Apple frameworks. SwiftUI, SwiftData, WidgetKit, ActivityKit, Swift Charts, StoreKit 2 — zero external dependencies.
