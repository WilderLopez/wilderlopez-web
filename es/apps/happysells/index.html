HappySells

Tu asistente de cambio de divisas, siempre visible

iOS 17+ iPadOS macOS watchOS Próximamente
App Store — Próximamente

HappySells convierte cualquier moneda o criptomoneda a tu moneda local en tiempo real. Soporta más de 240 países con detección automática de tu región y fuentes de datos especializadas por mercado. Integrada profundamente con el sistema operativo: widgets interactivos, Dynamic Island, complicaciones de Apple Watch y MenuBar en macOS. Gratis con suscripción Pro opcional.

Características

  • Conversión en Tiempo Real - Fiat (USD, EUR, CAD, MXN) y crypto (BTC, USDT, TRX) con tasas en vivo
  • Offline-First - Tasas en caché funcionan sin internet; indicador de frescura (verde/amarillo/rojo)
  • Historial de Transacciones - Historial completo con SwiftData, búsqueda y filtros
  • Compartir Ventas - Comparte como texto o tarjeta visual con marca de agua
  • Modo Privacidad - Oculta todos los números con un solo toque
  • Detección Automática de País - Selector de región automático con soporte para 240+ países
  • Moneda Base Configurable - Establece tu moneda base preferida
  • Sonido y Hápticos Premium - Efectos de sonido inmersivos y retroalimentación háptica
  • Refresh Inteligente - Actualización inteligente de tasas para conservar ancho de banda

Pro — $1.99/mes o $14.99/año

  • Widgets Ilimitados - Widgets en Home Screen, Lock Screen y StandBy
  • Live Activities - Dynamic Island con seguimiento de tasas en tiempo real
  • MenuBar en macOS - Acceso rápido via MenuBarExtra
  • Gráficas de Tendencia - Historial y tendencias de tasas con Swift Charts
  • Alertas de Precio - Notificaciones cuando las tasas alcanzan tu objetivo
  • Historial Ilimitado + Exportar CSV - Historial completo con exportación
  • Dashboard de Ganancias - Seguimiento de ganancias para traders
  • Complicaciones watchOS - Múltiples complicaciones para Apple Watch

Fuentes de Datos

  • Cuba - elToque API (tasas del mercado informal)
  • Crypto Global - CoinGecko API
  • Fiat Global - ExchangeRate API

Construida con 100% frameworks nativos de Apple. SwiftUI, SwiftData, WidgetKit, ActivityKit, Swift Charts, StoreKit 2 — cero dependencias externas.
