Política de Privacidad

HappySells

Última actualización: Marzo 2026

Introducción

Esta Política de Privacidad describe cómo HappySells ("nosotros", "nuestro" o "nos") maneja la información cuando usas nuestra aplicación de cambio de divisas para iOS, iPadOS, macOS y watchOS.

Almacenamiento de Datos

HappySells está diseñada con la privacidad como principio fundamental. Tu historial de transacciones y configuración personal se almacenan:

  • Localmente en tu dispositivo usando SwiftData con encriptación de iOS Data Protection
  • Los tokens de API se almacenan de forma segura en el Keychain del dispositivo

No operamos servidores que almacenen tus datos personales. Tu historial de transacciones y datos de conversión nunca salen de tu dispositivo.

Información que Recopilamos

Lo que NO Recopilamos

  • Tu historial de transacciones o montos de conversión
  • Información financiera o detalles de cuentas
  • Tu ubicación geográfica (la detección de región se hace en el dispositivo)
  • Datos de identificación personal

Lo que PODEMOS Recopilar (Opcional)

  • Reportes de errores anonimizados para mejorar la estabilidad de la app
  • Análisis básicos de uso (uso de funciones, no contenido ni montos)

Puedes desactivar los análisis en Configuración.

Solicitudes de Red

HappySells se conecta a APIs de terceros para obtener tasas de cambio:

  • elToque API - Para tasas del mercado informal cubano
  • CoinGecko API - Para tasas de criptomonedas
  • ExchangeRate API - Para tasas de monedas fiat globales

Estas solicitudes solo transmiten el par de divisas consultado. No se envía información personal, identificadores de dispositivo ni historial de transacciones a estos servicios. La app usa un enfoque offline-first — las tasas se cachean localmente y las solicitudes de red se minimizan con límites de refresh inteligentes.

Suscripciones

Las suscripciones Pro se procesan completamente a través de StoreKit y el App Store de Apple. No tenemos acceso a tu información de pago. Apple maneja toda la facturación y el procesamiento de pagos.

Seguridad de Datos

  • Todos los datos encriptados en reposo usando iOS Data Protection
  • Tokens de API almacenados en Keychain
  • Sin almacenamiento de datos personales en servidores
  • Conectividad de red monitoreada via NWPathMonitor (solo en dispositivo)
  • Cero dependencias externas — 100% frameworks nativos de Apple

Tus Derechos

Tienes control total sobre tus datos:

  • Eliminar transacciones individuales o todos los datos en cualquier momento
  • Exportar tu historial de transacciones como CSV
  • Desinstalar la app para eliminar todos los datos locales
  • Optar por no participar en análisis en Configuración

Privacidad de Menores

HappySells no está destinada a niños menores de 13 años. No recopilamos intencionalmente información de menores.

Cambios a Esta Política

Podemos actualizar esta Política de Privacidad de vez en cuando. Te notificaremos de cualquier cambio publicando la nueva política en esta página y actualizando la fecha de "Última actualización".
