Kodetime

Tu compañero de estimación con IA

macOS 14+
Próximamente

Kodetime es una aplicación nativa de macOS que usa Claude AI para analizar proyectos de software y proporcionar estimaciones precisas de tiempo para tareas de desarrollo. Deja de adivinar, empieza a estimar.

Características

  • Estimaciones con IA - Claude AI analiza tareas basándose en el contexto de tu proyecto
  • Gestión de Proyectos - Gestiona múltiples proyectos de software con detección automática de stack tecnológico
  • Integración GitHub/GitLab - Importa issues directamente vía OAuth
  • Desglose Detallado - Subtareas, factores de complejidad y niveles de confianza
  • Gestión de Tickets - Crea, organiza y rastrea tickets con 6 tipos y estados
  • Historial de Estimaciones - Rastrea la precisión con tiempo estimado vs real
  • macOS Nativo - Construido con SwiftUI para una experiencia Mac perfecta

Actualmente en desarrollo para macOS. Mantente atento a las actualizaciones.
