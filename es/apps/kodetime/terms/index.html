Términos y Condiciones

Kodetime

Última actualización: Febrero 2026

1. Aceptación de los Términos

Al descargar, instalar o usar Kodetime, aceptas estar sujeto a estos Términos y Condiciones. Si no estás de acuerdo con estos términos, por favor no uses nuestra app.

2. Descripción del Servicio

Kodetime es una aplicación de estimación de tareas con IA para macOS que ayuda a los desarrolladores a estimar tickets de proyecto usando inteligencia artificial. El servicio incluye:

  • Estimación de tiempo de tareas con IA usando Claude AI
  • Gestión de proyectos y seguimiento de tickets
  • Integración con GitHub y GitLab para importar issues
  • Detección de stack tecnológico
  • Almacenamiento local de datos con SwiftData

3. Requisitos de Clave API

Kodetime requiere que proporciones tu propia clave API de Anthropic para usar las funciones de estimación con IA:

  • Eres responsable de obtener y mantener tu clave API de Anthropic
  • Los cargos por uso de API de Anthropic son tu responsabilidad
  • Debes cumplir con los términos de servicio de Anthropic
  • Tu clave API se almacena de forma segura en tu Keychain local de macOS

4. Integraciones de Terceros

GitHub / GitLab

Cuando te conectas a GitHub o GitLab:

  • Autorizas a Kodetime a acceder a los repositorios que selecciones
  • Debes cumplir con los términos de servicio de GitHub y GitLab
  • Los tokens OAuth se almacenan en tu Keychain local
  • Puedes desconectar las integraciones en cualquier momento

5. Responsabilidades del Usuario

Eres responsable de:

  • Mantener la confidencialidad de tus claves API
  • La precisión de la información que proporcionas a la app
  • Asegurar que tienes autorización para acceder a los repositorios conectados
  • Usar la app en cumplimiento con las leyes aplicables

6. Descargo de Responsabilidad sobre Estimaciones con IA

Las estimaciones de tiempo generadas por IA se proporcionan solo como guía. No garantizamos:

  • La precisión de ninguna estimación de tiempo
  • Que las estimaciones coincidan con el tiempo real de desarrollo
  • Que las estimaciones sean adecuadas para planificación de proyectos o facturación

Las estimaciones de IA deben ser revisadas y validadas por desarrolladores experimentados antes de usarse en planificación de proyectos o comunicaciones con clientes.

Kodetime y su contenido original, características y funcionalidad son propiedad de Wilder Lopez y están protegidos por las leyes de propiedad intelectual. Tus datos de proyecto y tickets siguen siendo de tu propiedad.
