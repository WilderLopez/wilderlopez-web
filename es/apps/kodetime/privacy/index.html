Política de Privacidad

Kodetime

Última actualización: Febrero 2026

Introducción

Esta Política de Privacidad describe cómo Kodetime ("nosotros", "nuestro" o "nos") recopila, usa y protege tu información cuando usas nuestra aplicación de estimación de tareas con IA para macOS.

Almacenamiento de Datos

Kodetime está diseñado con la privacidad como prioridad. Tus datos se almacenan:

  • Localmente en tu Mac usando SwiftData (base de datos segura de Apple)
  • Claves API en Keychain usando almacenamiento seguro de macOS

No operamos servidores que almacenen tus datos de proyecto o tickets.

Información que Recopilamos

Lo que NO Recopilamos

  • Tu código fuente
  • Tus archivos de proyecto
  • Tus claves API (almacenadas solo en tu Keychain local)
  • Tus credenciales de GitHub/GitLab

Lo que PODEMOS Recopilar (Opcional)

  • Reportes de errores anonimizados
  • Análisis básicos de uso (uso de funciones, no contenido)

Servicios de Terceros

Anthropic (Claude AI)

Cuando solicitas una estimación de tarea, Kodetime envía lo siguiente a la API de Anthropic:

  • Título y descripción del ticket
  • Stack tecnológico del proyecto (detectado automáticamente)
  • No se envía código fuente a menos que esté explícitamente incluido en la descripción del ticket

Tu clave API de Anthropic se almacena localmente en tu Keychain de macOS y se envía directamente a Anthropic. Nunca vemos ni almacenamos tu clave API.

Por favor revisa la Política de Privacidad de Anthropic.

GitHub / GitLab

Si conectas tu cuenta de GitHub o GitLab:

  • Los tokens OAuth se almacenan en tu Keychain local
  • Solo accedemos a los repositorios que autorices
  • Los datos de issues se importan a tu base de datos local
  • No almacenamos tus credenciales en ningún servidor

Revisa la Política de Privacidad de GitHub y la Política de Privacidad de GitLab.

Análisis de Proyectos

Cuando agregas un proyecto, Kodetime analiza la carpeta para detectar:

  • Stack tecnológico (lenguajes, frameworks)
  • Estructura del proyecto

Este análisis se realiza completamente en tu Mac. No se suben ni transmiten archivos del proyecto.

Seguridad de Datos

  • Todos los datos almacenados localmente usando características de seguridad de macOS
  • Claves API y tokens OAuth almacenados en Keychain (encriptado)
  • Sin almacenamiento en la nube ni sincronización
  • Los datos permanecen en tu Mac

Tus Derechos

Tienes control total sobre tus datos:

  • Eliminar proyectos y tickets en cualquier momento
  • Desconectar cuentas de GitHub/GitLab
  • Remover claves API del Keychain
  • Desinstalar la app para eliminar todos los datos

Privacidad de Menores

Kodetime está diseñado para desarrolladores de software profesionales. No está destinado para menores de 13 años.

Cambios a Esta Política

Podemos actualizar esta Política de Privacidad de vez en cuando. Te notificaremos de cualquier cambio actualizando la fecha de "Última actualización".
