Privacy Policy

Kodetime

Last updated: February 2026

Introduction

This Privacy Policy describes how Kodetime ("we", "our", or "us") collects, uses, and protects your information when you use our AI-powered task estimation application for macOS.

Data Storage

Kodetime is designed with privacy as a priority. Your data is stored:

  • Locally on your Mac using SwiftData (Apple's secure database)
  • API keys in Keychain using macOS secure storage

We do not operate servers that store your project data or tickets.

Information We Collect

What We DO NOT Collect

  • Your source code
  • Your project files
  • Your API keys (stored only in your local Keychain)
  • Your GitHub/GitLab credentials

What We MAY Collect (Optional)

  • Anonymized crash reports
  • Basic usage analytics (feature usage, not content)

Third-Party Services

Anthropic (Claude AI)

When you request a task estimation, Kodetime sends the following to Anthropic's API:

  • Ticket title and description
  • Project technology stack (auto-detected)
  • No source code is sent unless explicitly included in ticket description

Your Anthropic API key is stored locally in your macOS Keychain and sent directly to Anthropic. We never see or store your API key.

Please review Anthropic's Privacy Policy.

GitHub / GitLab

If you connect your GitHub or GitLab account:

  • OAuth tokens are stored in your local Keychain
  • We access only the repositories you authorize
  • Issue data is imported to your local database
  • We do not store your credentials on any server

Review GitHub's Privacy Policy and GitLab's Privacy Policy.

Project Analysis

When you add a project, Kodetime analyzes the folder to detect:

  • Technology stack (languages, frameworks)
  • Project structure

This analysis is performed entirely on your Mac. No project files are uploaded or transmitted.

Data Security

  • All data stored locally using macOS security features
  • API keys and OAuth tokens stored in Keychain (encrypted)
  • No cloud storage or synchronization
  • Data remains on your Mac

Your Rights

You have full control over your data:

  • Delete projects and tickets at any time
  • Disconnect GitHub/GitLab accounts
  • Remove API keys from Keychain
  • Uninstall the app to remove all data

Children's Privacy

Kodetime is designed for professional software developers. It is not intended for children under 13.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by updating the "Last updated" date.

Contact Us

For questions about this Privacy Policy, contact us at: info@wilderlopezelias.com
